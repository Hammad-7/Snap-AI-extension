Privacy Policy

SnapAI Chrome Extension

Effective: February 25, 2026

SnapAI is built with privacy at its core. We have no servers, collect no data, run no analytics, and never see your API keys or the text you analyze. Everything stays on your device and between you and your chosen AI provider.

Overview

SnapAI is a Chrome extension that lets you select text on any webpage and analyze it using AI. It works entirely on your device — there are no SnapAI-operated servers involved. This policy explains what data the extension accesses, how it's used, and where it goes.

Data Collection

SnapAI does not collect, store, or transmit any personal data to us. We do not operate any servers or backend services. Specifically:

  • No analytics or telemetry of any kind
  • No user tracking or fingerprinting
  • No cookies or third-party scripts
  • No data shared with SnapAI or any third party
  • No account creation or sign-in required

API Keys

To use SnapAI, you provide an API key from your chosen AI provider. Your API keys are:

  • Stored locally on your device using chrome.storage.local
  • Never transmitted to any SnapAI server (we have none)
  • Never logged, recorded, or accessible to anyone but you
  • Used only to authenticate requests to your chosen AI provider

Text Processing

When you select text and trigger an action (Explain, Summarize, Fix Grammar, or Key Points), the selected text is sent directly from your browser to the API of the provider you have configured:

Supported AI Providers
  • OpenAI — api.openai.com
  • Anthropic — api.anthropic.com
  • Google Gemini — generativelanguage.googleapis.com
  • DeepSeek — api.deepseek.com

Your text is sent only to the single provider you have selected. No text is ever sent to SnapAI or to any provider you haven't chosen. Each provider has its own privacy policy and terms of service governing how they handle the data you send to their API.

Browser Permissions

SnapAI requests the following browser permissions, each for a specific purpose:

storage
Saves your API keys, selected provider, model choice, and preferences locally on your device. Nothing is synced externally.
activeTab
Reads the text you have selected on the current tab when you trigger an AI action. Only accesses the active tab, only when you interact with SnapAI.
sidePanel
Opens the SnapAI side panel for full conversation view with follow-up support, without leaving your current page.
contextMenus
Adds SnapAI actions to the right-click menu so you can analyze selected text directly from the context menu.
Host permissions (API endpoints)
Allows the extension to send your selected text to the AI provider APIs listed above. Requests are only made to the provider you have configured — no other network requests are made.

Data Storage

All data is stored locally using Chrome's chrome.storage.local API:

  • API keys for your configured providers
  • Your selected provider and model
  • Extension preferences

This data never leaves your device. It is not synced to any cloud service or Chrome sync.

Data Deletion

You are in full control of your data:

  • Uninstalling the extension automatically removes all stored data
  • You can clear your API keys at any time from the extension's settings page
  • No data persists on any external server because none exists

Third-Party Services

SnapAI uses Google Fonts (fonts.googleapis.com) to load the Outfit and DM Sans typefaces for the extension interface. This is the only external resource loaded besides the AI provider APIs you explicitly configure. No other third-party services, scripts, or trackers are used.

Children's Privacy

SnapAI does not knowingly collect any information from children under 13. The extension does not collect information from anyone — it operates entirely locally.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated effective date. Significant changes will be communicated through the extension's update notes on the Chrome Web Store.

Contact

If you have questions about this privacy policy or SnapAI's data practices, please open an issue on our GitHub repository or contact us through the Chrome Web Store support page.
